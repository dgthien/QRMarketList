QRMarketList

O melhor aplicativo para controlar suas compras.

O QRMarketList foi criado para te auxiliar a controlar suas compras. Confira abaixo algumas das inumeras vantagens que você ganha utilizando o QRMarketList:

  • Leitura de QRCode

    Com uma simples leitura do QRCode é possivel adicionar o produto a sua lista.

  • Leitura Código de Barras

    Além da leitura por QRCode tambem pode ser feita a leitura por código de barras.

  • Acesso por celular

    Acesse o QRMarketList pelo seu dispositivo mobile.

  • Comparador de Preços

    Realiza a compração de preços entre os vários estabelecimentos associados.
